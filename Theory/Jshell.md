# JShell

**JShell** is Java’s interactive shell (REPL). It was introduced in **Java 9**.

**REPL** stands for **Read–Eval–Print Loop** — you type code, JShell runs it immediately, and prints the result. You do not need a full class, `main` method, or separate compile/run steps.

## Start JShell

Open **Command Prompt** or **Terminal** and run:

```bash
jshell
```

You should see something like:

```text
|  Welcome to JShell -- Version 25
|  For an introduction type: /help intro
```

## Useful Commands

| Command       | Description                  |
| ------------- | ---------------------------- |
| `/help intro` | Short introduction to JShell |
| `/exit`       | Leave JShell                 |

## When to Use It

Use JShell to try out an API or method quickly, check the output, and then write the full program with confidence.
